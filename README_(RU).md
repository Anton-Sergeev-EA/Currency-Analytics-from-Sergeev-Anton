# Currency Analytics from Sergeev Anton.
Продвинутая система анализа валют с прогнозированием на основе машинного обучения, AI-ассистентом на базе RAG и 
интерактивным веб-интерфейсом.

## Описание.
Currency Analytics from Sergeev Anton - это полнофункциональное веб-приложение для анализа и прогнозирования курсов валют с 
использованием официальных данных Центрального Банка России. Система применяет ансамблевые модели машинного обучения 
(LightGBM, XGBoost, Random Forest, Gradient Boosting) для формирования прогнозов с доверительными интервалами. 
Включает AI-ассистента на основе RAG для ответов на вопросы пользователей и интерактивный веб-интерфейс с графиками 
в реальном времени.


## Возможности.
- Исторические данные курсов валют из API ЦБ РФ (USD/RUB, EUR/RUB).
- Ансамблевые ML-модели для прогнозирования с оценкой неопределенности.
- AI-ассистент на основе RAG для ответов на вопросы о валютах.
- Интерактивный веб-интерфейс с графиками и статистикой в реальном времени.
- RESTful API с документацией OpenAPI (Swagger UI.
- Локальная система кэширования (Redis опционально).
- Структурированное JSON-логирование.
- Поддержка Docker-контейнеризации.

## Установка.
### Системные требования.
- Python 3.12 или выше.
- pip - менеджер пакетов.

### Локальная.
1. Клонирование репозитория:
   git clone <repository-url>
   cd currency-analyzer
2. Создание и активация виртуального окружения:
   python -m venv venv
   source venv/bin/activate  # Linux/Mac.
   venv\Scripts\activate     # Windows.
3. Установка зависимостей:
   pip install -r requirements.txt
4. Настройка конфигурации:
   cp .env.example .env
5. Обучение ML-моделей:
   python train_models.py
6. Запуск приложения:
   python -m src.main
7. Открытие в браузере:
   http://localhost:8000/

### Установка через Docker.
   docker-compose up -d

## Использование.
### Веб-интерфейс.
Откройте в браузере:
http://localhost:8000/
Веб-интерфейс предоставляет:
- Статистика в реальном времени - Текущие курсы и волатильность.
- Интерактивные графики - Визуализация исторических данных.
- Выбор периода - 30, 90, 180, 365 дней.
- AI-ассистент - Чат-интерфейс для вопросов о валютах.
- Таблица прогнозов - Прогноз курсов на 7 дней.

## API Эндпоинты.
Метод.	       Эндпоинт.	       Описание.
GET	           /	               Интерактивный веб-интерфейс.
GET	           /health	           Проверка работоспособности системы.
GET	           /api/data	       Исторические курсы валют.
GET	           /api/forecast	   Прогноз курсов валют.
POST	       /api/ask	           Запрос к AI-ассистенту.
GET	           /api/stats	       Статистика по данным.
POST	       /api/refresh	       Обновление кэшированных данных.

## Интерактивная документация.
Откройте в браузере:
http://localhost:8000/api/docs

## Примеры API-запросов.
- Получение прогноза для USD:
curl "http://localhost:8000/api/forecast?days=7&currency=usd_rate"
- Запрос к AI-ассистенту:
curl -X POST http://localhost:8000/api/ask \
  -H "Content-Type: application/json" \
  -d '{"question": "Какой прогноз по доллару на следующую неделю?"}'
- Получение исторических данных:
curl "http://localhost:8000/api/data?period_days=30"
- Получение статистики:
curl "http://localhost:8000/api/stats"

## Примеры вопросов для AI-ассистента.
- "Какой прогноз по доллару на следующую неделю?"
- "Какой прогноз по евро?"
- "Сколько я заработаю, если вложу 100 000 рублей в евро?"
- "Сравни доллар и евро"
- "Какая валюта лучше для инвестиций?"
